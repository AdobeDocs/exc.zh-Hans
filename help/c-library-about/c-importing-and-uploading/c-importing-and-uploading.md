# 导入和上传文件{#import-and-upload-files}

从 Dropbox 或 Creative Cloud 导入文件，并通过您的设备上传文件，以便在 Adobe Experience Cloud 库中的某个位置管理和存储这些文件。

必须先设置文件夹，然后才能导入或上传文件。
