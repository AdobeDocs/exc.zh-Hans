# 导入和上传文件{#import-and-upload-files}

从Dropbox或Creative Cloud导入文件，并从您的Devicce上传文件，以便在Adobe Experience Cloud Library中的一个位置管理和存储这些文件。

必须先设置文件夹，然后才能导入或上传文件。
