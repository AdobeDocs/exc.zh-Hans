# 筛选资产{#filter-assets}

您可以在 Adobe Experience Cloud 库中筛选资产。

要在 Experience Cloud 库中筛选资产，请执行以下操作：

1. 单击漏斗图标以筛选资产。

   ![](assets/library_filter_assets.png)

1. 根据您希望筛选资产的方式，选取相应的选项。
