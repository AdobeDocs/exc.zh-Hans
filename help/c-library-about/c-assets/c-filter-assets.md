# 筛选资产{#filter-assets}

您可以在Adobe Experience Cloud Library中筛选资产。

要在Experience Cloud Library中筛选资产，请执行以下操作：

1. 单击漏斗图标以筛选资产。

   ![](assets/library_filter_assets.png)

1. 选择要筛选资产的选项。
