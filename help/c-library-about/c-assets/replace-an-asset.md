# 替换资产{#replace-an-asset}

您可以替换Adobe Experience Cloud Library中的资产。

要替换Experience Cloud Library中的资产，请执行以下操作：

1. 单击资产。
1. 单击资产旁边的 **[!UICONTROL “更多选项]** ”菜单(省略号)。

   ![](assets/library_asset_options.png)

1. 单击 **[!UICONTROL 替换]**。
1. 选择新资产以替换旧资产。
