# 将资产置入垃圾桶{#trash-an-asset}

您可以将 Adobe Experience Cloud 库中的资产置入垃圾桶，以便把它从库中删除。

要在 Experience Cloud 库中将资产置入垃圾桶，请执行以下操作：

1. 单击一个或多个要置入垃圾桶的资产。![](assets/import_options_mulit_select_trash.png)

1. 单击页面顶部的 **[!UICONTROL 垃圾桶]** 图标。
