# 资产{#assets}

跨Adobe Experience Cloud Library中的各种工具管理资产，轻松找到详细信息和版本并对资产采取措施。

在Experience Cloud Library中，您可以：

* 在大图标视图中预览资产或查看详细信息列表视图中的详细信息。
* 查看各个资产的详细信息。
* 替换资产。
* 查看并还原到资产的先前版本。
* 回收资产。
* 恢复之前删除的资产。
* 永久删除已删除的资产。
