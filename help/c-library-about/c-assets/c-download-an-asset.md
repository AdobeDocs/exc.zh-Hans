# 下载资产{#download-an-asset}

您可以在 Adobe Experience Cloud 库中下载资产。

1. 单击资产。
1. 单击资产旁边的&#x200B;**[!UICONTROL 更多选项]**&#x200B;菜单（省略号）。

   ![](assets/library_asset_options.png)

1. 单击&#x200B;**[!UICONTROL 下载]**&#x200B;即可下载资产。
