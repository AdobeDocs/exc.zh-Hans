# 下载资产{#download-an-asset}

您可以在Adobe Experience Cloud Library中下载资产。

1. 单击资产。
1. 单击资产旁边的 **[!UICONTROL “更多选项]** ”菜单(省略号)。

   ![](assets/library_asset_options.png)

1. 单击 **[!UICONTROL 下载]** 以下载资产。
